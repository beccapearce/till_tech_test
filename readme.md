#User Stories

```
As a worker
So that I can know the cost of an item
I want to be able to input and item and retrieve a cost
```

```
As worker
So that I do not have to duplicate my item inputs
I want to be able to add a quantity
```

```
As a worker
So that I can know the total cost of the items
I want to be able to calculate a total
```

```
As a worker
So that I can know the payable total
I want to add tax
```
